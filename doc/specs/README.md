## 如何创建新规格

1. 确定模块/功能名称（kebab-case）
2. 按模板结构创建 `specs/<module-name>.md`
3. 更新本文件的导航表
4. 更新 `doc/README.md` 的文档计数

## 状态生命周期

- `draft` → `in-review` → `approved`
