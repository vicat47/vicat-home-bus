## How to Use

### Creating a New Document

1. Determine the target subdirectory and navigate to it
2. Copy template: `cp ../.TEMPLATE.md NNN-title.md`
3. Name with numbering: `NNN-kebab-case-title.md`
4. Fill in content following the template structure
5. Update this README's navigation table

### Status Changes

- **Draft** → **Published**: Document is complete and reviewed
- **Published** → **Deprecated**: Document is no longer applicable
- **Published** → **Superseded by XXX**: Replaced by a newer document
